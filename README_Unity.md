# 镜面艺术：晶核竞技场 · Crystal Arena（Unity / C# 版）

一个使用 **Unity + C#** 实现的 2D 塔防课程项目。游戏以传统塔防为基础，引入 **镜面反射、棱镜分光与激光光路规划** 作为核心机制，使玩家不仅需要布置防御塔，还需要主动设计激光传播路径。

## 项目定位

- 项目类型：Unity 2D 塔防游戏
- 课程主题：计算机图形学课程大作业
- 核心特色：镜子与棱镜不只是装饰，而是直接影响胜负的创新玩法
- 展示重点：光线反射、折射、分光、程序化建模、程序化音频、战斗反馈与完整 HUD

## 核心创新

### 1. 镜面反射驱动的策略玩法

- 反射镜可以连续旋转，实时改变激光传播方向
- 同一座激光炮台可以借助镜子覆盖原本无法直射的路径区域
- 玩家不仅要考虑“塔放在哪里”，还要考虑“光往哪里走”

### 2. 棱镜分光带来的范围扩展

- 激光进入棱镜后会分裂为多束彩色光线
- 单束直射输出可扩展为多方向覆盖
- 光线路径变化清晰可见，具有很强的图形学展示效果

### 3. 从普通塔防到“光路构造”

- 传统塔防更偏向数值叠加与射程覆盖
- 本项目增加了“布置防御塔 + 设计光路 + 调整装置”的双层策略
- 镜子与棱镜共同构成本项目最重要的创新玩法来源

## 主要内容

### 1. 战斗与系统

- 四类主要防御塔：激光炮台、机枪台、加农炮、冰冻台
- 两类光学装置：反射镜、分光棱镜
- 15 波敌人推进，最终 Boss 为“深渊巨龙”
- 每波结束可进行 3 选 1 卡牌强化

### 2. 视觉与交互

- 暗色主题现代化 HUD 与右侧建造面板
- 激光脉动宽度、粒子命中、爆炸、核心光环、漂移雾层
- Boss 出场、抽卡、核心受损带有震屏与短暂停顿
- 胜利/失败界面展示完整统计数据，适合作业答辩展示

### 3. 程序化内容

- 程序化精灵：圆形、方形、三角形、环形、软发光圆、菱形
- 程序化角色造型：暗影游灵、幽光精灵、铁甲魔像、深渊巨龙
- 程序化音效：部署、射击、反射、击杀、Boss 吼叫等
- 程序化背景音乐：运行时生成旋律、低音与节奏

## 运行环境

- Unity 6.4（6000.4.11f1）或 Unity 6 / 2022 LTS
- 渲染管线：Built-in 或 URP
- 建议使用 2D 模板打开项目

## 运行方式

1. 使用 Unity Hub 打开当前项目文件夹
2. 打开场景 `Assets/Scenes/Main.unity`
3. 点击 `Play` 运行游戏

## 主要脚本结构

```text
Assets/Scripts/
  Defs.cs          全局配置、塔与敌人定义、路径点、颜色配置
  Game.cs          游戏主控、输入处理、UI、统计、激光渲染
  GameMap.cs       地图网格、路径判定、背景与氛围表现
  WaveManager.cs   波次构建、刷怪节奏、难度推进
  Enemy.cs         敌人移动、受伤、死亡、程序化动画
  Tower.cs         防御塔逻辑、升级、移动、激光索敌
  Mirror.cs        镜面旋转与反射法线
  Prism.cs         棱镜折射与分光参数
  LaserSystem.cs   激光追踪、多段反射、折射与分光
  Projectile.cs    普通弹道与命中逻辑
  ParticleBit.cs   粒子特效
  CardSystem.cs    波间抽卡、增益效果、Buff HUD
  MusicManager.cs  程序化背景音乐
  SfxManager.cs    程序化音效
  SpriteFactory.cs 程序化精灵工厂
```
